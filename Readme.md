# DevConf 2026 — Conference Landing Page

A modern developer conference landing page built with pure HTML5 and CSS3.

## 🔗 Live Demo
[View Live Site](https://mdsaimon0114.github.io/DevConf-2026/)

## 🛠️ Built With
- HTML5
- CSS3
- Google Fonts (Inter)

## 📌 Features
- Navbar with smooth hover effects
- Full-width Hero Banner with dark overlay
- Speaker Cards with 2x2 grid layout
- Pricing Section with highlighted Pro plan
- Conference Tracks Section
- Event Stats Section
- Footer with social media links
